# Sistem za rezervaciju termina - Školski projekat

Ovo je web aplikacija za rezervaciju termina razvijena kao projekat iz predmeta Web programiranje. Aplikacija omogućava korisnicima da se registruju, prijave i upravljaju svojim terminima.

## Korištene Tehnologije
* **Frontend:** HTML, CSS, JavaScript (Vanilla)
* **Backend:** PHP
* **Baza podataka:** MySQL (XAMPP)
* **Autentifikacija:** Firebase Authentication

## Funkcionalnosti
* **Login/Registracija:** Sigurna prijava putem Firebase servisa.
* **CRUD operacije:** Dodavanje i brisanje termina iz lokalne baze.
* **Lokalni server:** Rad sa PHP i MySQL bazom preko XAMPP panela.

## Kako pokrenuti projekat
1. Instalirati **XAMPP** i pokrenuti Apache i MySQL module.
2. Folder sa projektom prebaciti u `C:/xampp/htdocs/`.
3. U **phpMyAdmin** napraviti bazu `rezervacije` i pokrenuti priloženi SQL kod.
4. Otvoriti preglednik i unijeti adresu: `http://localhost/rezervacije/login.html`.
